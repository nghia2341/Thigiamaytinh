# Smart Security System

## Giới thiệu
Dự án xây dựng hệ thống giám sát an ninh thông minh phát hiện người xâm nhập vào vùng cấm bằng YOLOv8 và OpenCV.

## Chức năng
- Nhận video từ webcam
- Phát hiện người theo thời gian thực
- Xác định vùng cấm
- Cảnh báo khi người đi vào vùng cấm
- Lưu ảnh sự kiện
- Ghi log thời gian phát hiện
- Phát âm thanh cảnh báo

## Công nghệ sử dụng
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy
